# 选择分发方法

镜像分发频谱：
- 托管注册表，公共仓库
- 托管注册表，私有仓库
- 私有注册表
- 自定义镜像分发架构
- 镜像源代码分发

选择标准：
- 成本
- 可见性
- 传输速度或带宽开销
- 生命周期控制
- 可用性控制
- 访问控制
- 工件完整性
- 工件机密性
- 必要的专业知识
